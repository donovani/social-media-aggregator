Folder Breakdown
----------------

Buddy-Buddy - This is the root folder, you are here.

	Code -This contains all the code to run Buddy-Buddy

		css - This contains all the css files for the webpage's design

		html - This contains all the html files used in Buddy-Buddy

		img - This contains some general Buddy-Buddy specific icons
			svg - This contains icons specific to social medias

		js - This contains all the javascript files for Buddy-Buddy
			twitter - This contains the twitter specific javascript files

		php - This contains all the php files for Buddy-Buddy
			twitter - This contains the twitter specific php files

		static - This contains more Buddy-Buddy specific image

	Documentation - This contains all the documentation that was completed through out the course of this project

		Documents Throughout - This contains files submitted throughout development such as the Design Doc, Project Plan, and Updates

		Final Documents - This contains the final documents to be submitted such as The Poster and Final Document

----------
How to Run
----------

1. Download and set up a WAMP/LAMP server

For WAMP:
	1. copy the 'Code' folder into your htdocs folder
	2. Start Apache
	3. Open a web browser and navigate to localhost/code
	4. Enjoy!

For LAMP:
	1. Copy the 'Code' folder into your /var/www/html/ directory
	2. Open a web browser and navigate to localhost/code or localhost/code/index.html
	3. Enjoy!
